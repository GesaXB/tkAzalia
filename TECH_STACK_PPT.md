# **Daftar Tech Stack & Tools - Proyek tkAzalia**

Berikut adalah rangkuman teknologi yang digunakan dalam proyek **tkAzalia** untuk materi Presentasi (PPT).

---

## **1. Core Framework & Language**
- **Next.js 16 (React 19)**: Framework utama untuk membangun website yang cepat, responsif, dan ramah SEO.
- **TypeScript**: Versi aman dari JavaScript yang membantu mencegah kesalahan pengetikan kode (*bugs*) saat pengembangan.

## **2. Database & Data Management**
- **PostgreSQL**: Sistem database relasional yang kuat untuk menyimpan data siswa, pendaftaran, dan konten sekolah secara terstruktur.
- **Supabase**: Layanan *cloud* yang menyediakan infrastruktur database PostgreSQL secara online agar aplikasi bisa diakses kapan saja.
- **Prisma ORM**: Jembatan antara kode aplikasi dengan database, memudahkan pengelolaan data tanpa harus menulis query SQL yang rumit.

## **3. Styling & Interface**
- **Tailwind CSS v4**: Framework desain untuk membuat tampilan website yang modern, bersih, dan profesional dengan sangat cepat.
- **Framer Motion**: Library animasi yang digunakan untuk memberikan efek transisi halus agar website terasa lebih interaktif dan "hidup".
- **Lucide React**: Set ikon yang bersih dan konsisten untuk antarmuka pengguna.
- **Recharts**: Library grafik untuk menampilkan statistik pendaftaran di dashboard admin.

## **4. Deployment & Hosting**
- **Vercel**: Platform infrastruktur yang digunakan untuk mempublikasikan website agar bisa diakses secara publik melalui internet.

---

### **Ringkasan Singkat (Satu Baris):**
> **Next.js (TS) + PostgreSQL (Supabase) + Prisma ORM + Tailwind CSS + Vercel.**
